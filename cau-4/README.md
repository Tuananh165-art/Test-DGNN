# University Network - React

Mạng lưới trường đại học Việt Nam với biểu đồ tương tác.

## Cài đặt

```bash
npm install
```

## Chạy

```bash
npm start
```

Mở http://localhost:3000

## Tính năng

- Biểu đồ mạng lưới tương tác
- Lọc theo vùng miền
- Tìm kiếm trường
- Thông tin chi tiết
- Thống kê

## Công nghệ

- React 18
- Vis-network
- Interactive graph
