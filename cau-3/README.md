# COVID-19 Dashboard - React

Dashboard hiển thị thống kê COVID-19 cho Việt Nam.

## Cài đặt

```bash
npm install
```

## Chạy

```bash
npm start
```

Mở http://localhost:3000

## Tính năng

- Biểu đồ đa dạng: Line, Area, Bar, Pie, Scatter
- Bộ lọc thời gian tự động cập nhật
- Tương tác: Hover, Zoom (scroll), Pan (drag)
- Responsive design
- Dữ liệu từ YouGov COVID-19 Tracker

## Công nghệ

- React 18
- Chart.js
- PapaParse
