# Lựa chọn 3: Quản lý chi tiêu cá nhân
# Expense Tracker - React

Ứng dụng quản lý chi tiêu cá nhân với React.

## Cài đặt

```bash
npm install
```

## Chạy

```bash
npm start
```

Mở http://localhost:3000

## Tính năng

- Thêm/Sửa/Xóa giao dịch
- Quản lý ngân sách
- Dashboard với biểu đồ
- Lưu trữ LocalStorage

## Công nghệ

- React 18
- Chart.js
- LocalStorage
